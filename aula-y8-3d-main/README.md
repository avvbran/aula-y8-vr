# Práctica 4.2: Creación de Escenarios VR - Aula Y8

## 🚀 Descripción del Proyecto
Este proyecto consiste en un visualizador web interactivo de Realidad Virtual (VR) que carga un modelo 3D del **Aula Y8**, diseñado previamente en *SketchUp 2025* y exportado en formato `.glb`.

Desarrollado utilizando:
* **HTML5, CSS3 y JavaScript (ES6+)**
* **Three.js** (v0.160.0) vía CDN para el renderizado 3D y compatibilidad con WebXR (VR).
* **Bootstrap 5** para el diseño responsivo del Navbar y Footer.
